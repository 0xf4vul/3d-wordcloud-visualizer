# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-25

### Added
- Initial release
- Support for JSON/MD/TXT file formats
- Automatic Chinese word segmentation
- Top 30 word frequency statistics
- 3D word cloud visualization with Three.js
- Mouse interaction (rotate, zoom, pan)
- Starry background effect
- Random color scheme for words
- Draggable panels with magnetic snap
- Snap icons with progressive disclosure
- Auto-rotate functionality
- Responsive UI
