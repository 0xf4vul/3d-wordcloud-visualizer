# 3D Word Cloud Visualizer 🌍

将对话历史或其他文本数据自动转换为炫酷的 3D 地球词云可视化效果。

**当前版本**: v1.2.0 | **安全评级**: ⭐⭐⭐⭐⭐ **A级（低风险）** | **安全评分**: 95/100

## 功能特性

### 核心能力

1. **多格式文件导入**
   
   - 支持 JSON / MD / TXT 三种格式文件
   - 自动提取文本内容，智能识别数据结构

2. **自动分词和词频统计**
   
   - 使用自定义正则表达式分词（无需外部库）
   - 自动统计词频，提取 TOP30 高频词
   - 智能过滤停用词
   - 完全本地处理，数据不上传云端

3. **3D 地球词云可视化**
   
   - Three.js 3D 渲染引擎
   - 真实的 3D 地球模型
   - 词云纹理映射到地球表面
   - 15 种颜色池循环分配，每个词颜色不同
   - 美丽的星空背景（5000 颗星星）

4. **交互功能**
   
   - 左键拖拽：旋转地球
   - 滚轮缩放：放大缩小
   - 右键拖拽：平移视角
   - 自动旋转展示（可暂停）
   - 面板拖拽和磁吸功能
   - 磁吸图标渐进式揭示（默认 opacity 0.3，悬停 1.0）

5. **友好提示系统**
   
   - 加载提示（旋转图标 + 文字说明）
   - 进度条动画（0% → 100%）
   - 成功弹窗（显示关键词数量、最高频词）
   - 错误弹窗（显示错误信息）
   - 4 秒自动消失

### 技术实现

- **Three.js** v0.160.0 - 3D 渲染引擎
- **OrbitControls** - 交互控制器
- **Segmentit** v2.0.3 - 中文分词库
- **Canvas 纹理映射** - 词云渲染
- **自定义滚动条样式** - 优化视觉体验

## 使用方法

1. 浏览器打开 HTML 文件
2. 点击"选择文件"按钮，上传 JSON/MD/TXT 文件
3. 等待自动解析（分词 + 词频统计）
4. 交互探索词云（拖拽旋转/滚轮缩放）

## 面板操作

- 拖拽面板到边框：自动磁吸
- 双击磁吸图标：恢复完整面板
- 右键地球：重置面板到初始状态

## 安全特性

### 安全等级

**安全评级**: ⭐⭐⭐⭐⭐ **A级（低风险）** | **安全评分**: 95/100

### 安全措施

- ✅ **SRI 完整性校验**：Three.js CDN 使用 SRI 防止投毒攻击
- ✅ **XSS 防护**：所有用户输入通过 `escHtml()` 函数转义
- ✅ **文件大小限制**：10MB 上限，防止恶意大文件攻击
- ✅ **超时保护**：文件解析和渲染超时保护（5 秒）
- ✅ **原型链保护**：词频统计使用 `Object.create(null)` 防止污染
- ✅ **纯前端处理**：数据不上传云端，完全本地处理
- ✅ **无外部 API 调用**：无敏感信息存储
- ✅ **编码规范**：UTF-8-sig 编码，避免中文乱码
