# 3D Word Cloud Visualizer

将对话历史或其他文本数据自动转换为炫酷的 3D 地球词云可视化效果。

## 功能特性

- **多格式支持**：JSON/MD/TXT
- **自动分词**：使用 Segmentit 中文分词库
- **词频统计**：TOP30 高频词
- **3D 可视化**：Three.js 渲染引擎
- **交互功能**：鼠标拖拽旋转、滚轮缩放、右键平移

## 使用方法

1. 准备 JSON/MD/TXT 文件
2. 在浏览器中打开生成的 HTML 文件
3. 上传文件，自动生成词云

## 技术栈

- Three.js v0.160.0
- OrbitControls
- Segmentit v2.0.3
