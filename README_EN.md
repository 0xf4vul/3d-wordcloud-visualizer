# 3D Word Cloud Visualizer

Transform conversation history or any text data into a stunning 3D word cloud visualization on a rotating globe.

## Features

- **Multiple Format Support**: JSON/MD/TXT
- **Automatic Segmentation**: Chinese word segmentation using Segmentit
- **Word Frequency Statistics**: Top 30 most frequent words
- **3D Visualization**: Three.js rendering engine
- **Interactive Controls**: Mouse drag to rotate, scroll to zoom, right-click to pan

## Usage

1. Prepare JSON/MD/TXT file
2. Open the generated HTML file in your browser
3. Upload file to automatically generate word cloud

## Tech Stack

- Three.js v0.160.0
- OrbitControls
- Segmentit v2.0.3
